# task 2-:Data Visualization

## 📌 Project Overview

This project focuses on exploring and visualizing the Iris dataset using Python and Matplotlib. The objective is to analyze the dataset through graphical representations, identify patterns, understand feature distributions, and discover relationships between different flower measurements.

---

## 🎯 Objective

- Load and inspect the Iris dataset.
- Explore the dataset structure and features.
- Create meaningful visualizations to identify patterns.
- Extract insights from the visualized data.

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

The project uses the **Iris Dataset**, which contains measurements of iris flowers, including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

---

## 🔍 Project Workflow

### 1. Data Loading
- Imported the required Python libraries.
- Loaded the Iris dataset into a Pandas DataFrame.

### 2. Data Exploration
- Displayed the first few records using `head()`.
- Examined dataset information using `info()`.
- Reviewed the dataset structure and data types.

### 3. Data Visualization
- Created a **Bar Chart** to show the distribution of different Iris species.
- Created a **Histogram** to visualize the distribution of Sepal Length values.
- Created a **Scatter Plot** to analyze the relationship between Sepal Length and Petal Length.

### 4. Data Analysis
- Interpreted each visualization to identify trends, feature distributions, and relationships between variables.
- Observed the positive correlation between Sepal Length and Petal Length.

---

## 📊 Key Outcomes

- Successfully visualized the distribution of Iris species.
- Identified the range and spread of Sepal Length values.
- Observed a positive relationship between Sepal Length and Petal Length.
- Demonstrated how visualization simplifies data exploration and interpretation.

---

## 📈 Key Insights

- The dataset shows a highly balanced class distribution across Iris species.
- There is a relationship between sepal length and petal length.
- By visualizing the dataset, it makes easier to identify trends, distribution and pattern that might not seem obvious through raw data.
